## Antarcticite
Wordplay involved with ant-arcticite.

Antarcticite is a gem of colorless acicular trigonal crystals. 

As its name implies, it was first described in 1965 for an occurrence in Antarctica where it occurs as crystalline precipitate from a highly saline brine in Don Juan Pond, in the west end of Wright Valley, Victoria Land. [ref](https://en.wikipedia.org/wiki/Antarcticite)
